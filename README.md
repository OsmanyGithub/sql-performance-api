# SQL Performance Dashboard

Compare optimized vs slow SQL queries and view the top N customers with a PostgreSQL database and Streamlit dashboard.

**Features**
- Top N customers by total spend
- Compare optimized vs slow query execution times
- Dockerized PostgreSQL + Streamlit setup
- [Watch project demo videos](YOUR_VIDEO_LINK_HERE)

**Tech Stack**
Python 3.10+, Streamlit, PostgreSQL, Docker, Pandas, Psycopg2

**Setup**
```bash
git clone git@github.com:OsmanyGithub/sql-performance-api.git
cd sql-performance-api
docker-compose up --build
